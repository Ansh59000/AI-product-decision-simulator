# AI-product-decision-simulator
An interactive tool that models real-world tradeoffs in AI systems (e.g., accuracy vs latency, precision vs recall). It allows users to simulate decisions, visualize their impact on key metrics, and understand how product choices affect user experience and business outcomes.
Instead of guessing outcomes, users can simulate decisions like:

Switching models
Reducing costs
Improving accuracy
Changing safety constraints

…and instantly see how those choices impact key product metrics.
